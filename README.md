# Hooky Bio

Profile bio generator for content creators. Create scroll-stopping bios for Instagram, TikTok, YouTube, X, and LinkedIn.

## Features

- Multi-platform support (Instagram, TikTok, YouTube, X, LinkedIn)
- Goal-based optimization (Followers, DMs, Sales, Authority)
- Style control (Minimal, Balanced, Expressive)
- Multiple output types (Hook, Bio, CTA, Variations, All)

## Setup

1. Clone the repository
2. Create a `.env` file based on `.env.example`
3. Add your OpenAI API key to the `.env` file
4. Deploy to Netlify

## Environment Variables

| Variable | Description |
|----------|-------------|
| `OPENAI_API_KEY` | Your OpenAI API key |

## Deployment

### Netlify

1. Connect your GitHub repository to Netlify
2. Add the `OPENAI_API_KEY` environment variable in Netlify dashboard
3. Deploy

## Tech Stack

- HTML/CSS/JavaScript (Vanilla)
- Netlify Functions (Serverless)
- OpenAI API (gpt-4o-mini)
